# Forviz Content Management Service #

### Setup ###

````
npm install
````


### Development ###

````
npm run dev
````

Server จะ start ที่ localhost:4000
และ Client จะ start ที่ localhost:3000

### Build for Deployment ###

````
npm run build
````
ระบบจะ compile และสร้าง folder /build ขึ้นมา

### Start (Production) ####
````
npm run start
````
